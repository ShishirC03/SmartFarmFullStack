# SmartFarm - AI Animal Intrusion Detection System

This is a new **React Native** + **Node.js** project.  
It provides real-time animal intrusion detection for farms using a mobile camera feed, a Node/Express backend and a YOLO-based detector. The mobile app receives live alerts via Socket.IO and shows a Detection History.
